# imooc-react-chat

> Redux+React Router+Node.js学习

运行项目<br>
1、先安装mongodb和环境搭建： [http://www.cnblogs.com/ccyinghua/p/7887713.html](http://www.cnblogs.com/ccyinghua/p/7887713.html)<br>
2、cnpm install<br>
3、node server/server.js 或者 nodemon server/server.js<br>
4、npm start<br>
